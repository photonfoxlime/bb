# Blooming Blockery

A tree-structured document editor that catalyzes text-based design and implementation.

## Installation

Check the [releases page](https://github.com/photonfoxlime/bb/releases) for the latest executable.

Alternatively, here's one-liner to run:

Npx
```shell
npx @miorin.blooming-blockery
```

Alternatively, using package managers:

<details>
<summary>Distributed Packages</summary>
Not published on crate.io due to binary size.

Arch Linux
```shell
pacman -S blooming-blockery
```

Homebrew
```shell
brew install blooming-blockery
```

Winget
```shell
winget install --id Miorin.BloomingBlockery
```

Npm
```shell
npm install -g @miorin.blooming-blockery
```
</details>




## Skills

The CLI for Blooming Blockery: `blooming-blockery-cli`
```shell
npx skills add https://github.com/photonfoxlime/bb --skill blooming-blockery-cli
```
